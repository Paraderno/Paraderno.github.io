明天不上课，今晚有场 Codeforce，想了想还是不打那一场，下午打个 VP 好了。

比赛

A.Password （0:04）

题意

从 $10-n$ 个数字中选 $2$ 个数字，这 $2$ 个数字各用两次，组成一个四位数（允许前导 $0$）。

求方案数。

分析

对于任意两个数字 $a$，$b$，能组成的方案为 $aabb$，$abab$，$abba$，$baab$，$baba$，$bbaa$ 共六种方案

方案总数 $ans = C^2_{10-n} \times 6$

Code

B. Permutation Value （0:18）

题意

对于长度为 $n$ 的一个排列，定义 $Value$ 为子段是排列的个数

构造一个排列，让它的 $Value$ 最小

分析

不难发现，$Value$ 最小为 $2$，即 $[1]$ 和 $[1,2,3,\cdots,n]$。

要满足条件，最简单的构造方式为 $[1,n,2,3,\cdots,n-1]$

Code

C. Save the Magazines （0:27）

题意

长度为 $n$ 的数组，权值为 $a_i$，权重为 $b_i(b_i = 0,1)$

循环 $i = 2 \to n$，当 $b[i - 1] = 0$ 且 $b[i] = 1$ 时，可以让 $b[i - 1] = 1$ 且 $b[i] = 0$

求 $\max(\sum_{i = 1 \to n} a_ib_i)$

分析

对于 $01$ 的情况，当 $a_0 > a_1$ 时，进行操作
对于 $01111$ 的情况，当存在一个 $j$ 满足 $a_0 > a_{1j}$ 时，进行操作

Code

D. Problem with Random Tests （1:17）

题意

给一个纯随机的 $01$ 字符串，从中选择两个子串，子串的价值 $Value$ 为将其视为二进制的值

求 $\max(Value1 | Value2)$

分析

毫无疑问，一定会取整个字符串，因为他一定是所有串中最大的，且最高位为 $1$
考虑寻找一个字符串能够尽可能的增大 $Value$，也就是从高位到低位，依次考虑这个位置的 $0$ 能否通过或运算让它变成 $1$

考虑到该字符串为纯随机，每增长一位，备选的字符串数量平均减少一半。

Code

E. FTL （1:52）

题意

两种武器，各有一个伤害 $p$ 与冷却时间 $t$。
一个敌人，有血量 $h$ 与护甲 $s$。

单独使用武器，能造成伤害 $p - s$。
同时使用两个武器，能造成伤害 $p_1 + p_2 - s$

求击败敌人的最短时间。

分析

预处理计算 $F[x]$ 为造成 $x$ 伤害的最短时间
于是问题转化为无穷背包问题。

对于 $F[x]$ 的计算，枚举两种武器各使用多少次，并且最后一次为同时使用。
花费时间为 std::max(t1 * i, t2 * j)
造成伤害为 (i - 1) * (p1 - s) + (j - 1) * (p2 - s) + (p1 + p2 - s)，某一个武器使用次数为 $0$ 单独考虑。

Code

总结

表现分：2187，橙名的水平。

总体表现还可以再优化一下。

$B$ 题题意理解出问题了，以为 $[2,3,4]$ 这种也是合法的。后面才反应过来必须从 $1$ 开始

$C$ 题差不多算读完题就会

$D$ 题想了好久，开始有看到那句纯随机，禁止 $hack$，但没想明白这有什么意义。
然后就往 $KMP$，$AC$ 自动机那方面去想，但又感觉难度不太对。
最后反应过来纯随机是为了什么。差点就开摆了。

$E$ 题第一反应想到了 $DP$，但没想到用 $F[x]$ 数组优化。
打算枚举 $(i,j)$，满足不存在 $(I,J)$ 是 $(i,j)$ 的整数倍。这样会优化一些选项（虽然还是 $10^7$ 数量级）
然后想到了 $F[x]$ 预处理。
只能说练习少了，对题目没打 $OI$ 时那么敏感了。
